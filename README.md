# SimpleCopyAnim
Copy animation from different scene or maya vesion

Copy python file to your script location the type this to script editor

When copy:

import CPAnimation as CP\
reload(CP)\
CP.CopyAnimation()

When paste:

import CPAnimation as CP\
reload(CP)\
CP.PasteAnimation()
